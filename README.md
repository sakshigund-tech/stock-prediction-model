# HORIZONS - Stock Price Predictor

## Overview

HORIZONS is a Machine Learning based Stock Price Prediction System developed to analyze historical stock market data and predict future stock price trends.

The project uses Deep Learning and data analysis techniques to help users understand market behavior and make informed trading decisions.

---

## Features

- Historical stock market data analysis
- Stock price prediction using Machine Learning
- Data preprocessing and transformation
- Visualization of stock trends
- User-friendly prediction workflow
- Support for technical indicators
- Real-time and historical data handling

---

## Technologies Used

- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- yFinance API

---

## Machine Learning Models

This project explores:

- LSTM (Long Short-Term Memory)
- GRU Networks
- Logistic Regression
- Time Series Forecasting

## Installation

Clone the repository:

```bash
git clone https://github.com/sakshigund-tech/stock-prediction-model.git
```

Go to the project folder:

```bash
cd stock-prediction-model
```

Install dependencies:

```bash
pip install -r requirements.txt
```
